# دليل التثبيت والتشغيل

## 📋 المتطلبات قبل البدء

تأكد من تثبيت:
- **Node.js** (الإصدار 12 أو أحدث) - [تحميل](https://nodejs.org/)
- **npm** (يأتي مع Node.js عادة)
- **Git** - [تحميل](https://git-scm.com/)

للتحقق من التثبيت، افتح Terminal/Command Prompt وشغّل:
```bash
node --version
npm --version
git --version
```

---

## 🚀 خطوات التثبيت

### الخطوة 1: استنساخ المشروع
```bash
git clone https://github.com/talkadeliveryth-pixel/yemen-medicine-bank-dhamar.git
cd yemen-medicine-bank-dhamar
```

### الخطوة 2: تثبيت المكتبات
```bash
npm install
```

سيستغرق هذا عدة دقائق. انتظر حتى ينتهي.

### الخطوة 3: إضافة شعار التطبيق
1. انسخ صورة شعار بنك الدواء
2. ضعها في المجلد: `public/logo.png`

الصيغ المدعومة: PNG, JPG, GIF

### الخطوة 4: تشغيل التطبيق

#### التشغيل العادي:
```bash
npm start
```

#### التشغيل مع المراقبة (للتطوير):
```bash
npm run dev
```

## 🌐 الوصول للتطبيق

## 🔧 الاستخدام الأساسي

### 1. إضافة مريض جديد
- اذهب إلى "إضافة مريض"
- املأ البيانات الشخصية
- أضف الأدوية المصروفة
- اضغط "حفظ البيانات"

### 2. عرض قائمة المرضى
- اذهب إلى "قائمة المرضى"
- ستجد جدول بجميع المرضى المسجلين

### 3. البحث عن مريض
- اذهب إلى "البحث"
- ابحث باسم المريض أو رقم البطاقة

### 4. تصدير البيانات
- اذهب إلى "استيراد/تصدير"
- اضغط "تحميل Excel" لتحميل ملف

### 5. استيراد البيانات
- اذهب إلى "استيراد/تصدير"
- حدد ملف Excel
- اضغط "استيراد"

## 🛑 حل المشاكل الشائعة

### المشكلة: لا يعمل `npm install`

**الحل:**
```bash
# احذف مجلد node_modules
rm -rf node_modules

# احذف ملف package-lock.json
rm package-lock.json

# جرب التثبيت من جديد
npm install
```

### المشكلة: المنفذ 3000 مستخدم بالفعل

**الحل:**
غيّر رقم المنفذ في `.env`:
```
PORT=3001
```

### المشكلة: قاعدة البيانات لا تعمل

**الحل:**
```bash
# احذف ملف قاعدة البيانات
rm medicine_bank.db

# شغّل التطبيق من جديد
npm start
```
